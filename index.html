3. Afbeeldingen toevoegen

Punten: 4

Opdracht

  • Maak een nieuwe branch: afbeeldingen
  • Controleer dat je in deze branch zit
  • Zoek een passende afbeelding voor dit hoofdstuk (id=afbeeldingen), voeg deze afbeelding toe in de map images.
  • Pas de src aan van de afbeelding (id=afbeelding) van dit hoofdstuk aan, zodat de nieuw toegevoegde afbeelding zichtbaar is.
  • Commit deze wijziging
  • Merge de aanpassingen van afbeeldingen naar master/main.
  • Push de master/main branch naar github

4. Een conflict oplossen

Punten: 8

Winnaar: Niemand

Ant-Man

  • Begin een nieuwe branch: ant_man.
  • Pas de hierboven aangegeven winnaar (id=winnaar) aan naar: Ant-Man.
  • Pas de src van de afbeelding (id=winnaar-image) aan naar: images/ant_man.png.
  • Commit deze wijziging.
  • Push de ant_man branch naar github.

Wasp

  • Begin een nieuwe branch: wasp.
  • Pas de hierboven aangegeven winnaar (id=winnaar) aan naar: Wasp.
  • Pas de src van de afbeelding (id=winnaar-image) aan naar: images/wasp.png.
  • Commit deze wijziging.
  • Push de wasp branch naar github.

Fight!

  • Ga terug naar de master/main branch.
  • Refresh de pagina en check of Niemand de winnaar is.
  • Merge de aanpassingen van wasp.
  • Controleer of Wasp de winnaar is (refresh de pagina).
  • Merge de aanpassingen van ant_man.
  • Conflict!

    Je hebt nu een conflict in de header van dit article.

  • Zorg dat Ant-Man de winnaar is.
  • Commit de merge
  • Push de master/main branch naar github

5. Een conflict oplossen

Punten: 8

Soms is er bij een conflict geen keuze, maar moet je handmatig de wijzigingen samen voegen.

Donaties: FTT 0,00

Rich Boy

  • Begin een nieuwe branch: rich_boy.
  • De rapper Rich Boy heeft net 187 FTX FTT coins naar onze Wallet overgemaakt.
    Pas het hierboven aangegeven donatiebedrag (id=donaties) aan.
  • Commit deze wijziging.
  • Push de rich_boy branch naar github

Mrutte

  • Begin een nieuwe branch: mrutte vanuit master/main
  • Mrutte is een beetje skeer, hij heeft FTT 1000 belooft,
    maar geeft ons slechts FTT 1.
    Pas het hierboven aangegeven donatiebedrag (id=donaties) aan.
  • Commit deze wijziging.
  • Push de mrutte branch naar github

Conflict!

  • Merge de aanpassingen van mrutte naar master/main.
  • Controleer of het donatie bedrag klopt (refresh de pagina).
  • Merge de aanpassingen van rich_boy naar master/main.
  • Conflict!
  • Zorg dat de donaties bij elkaar worden opgeteld.
  • Controleer of het donatiebedrag correct is (refresh de pagina).
  • Commit de merge
  • Push de master/main branch naar github
